# Secure Django Task Manager & API

A full-stack Django application featuring a multi-user task management system and a RESTful API backend.

## 🚀 Features
- **Multi-User Authentication:** Secure signup/login using Django's built-in auth.
- **Data Privacy:** Users can only view, edit, or delete their own tasks.
- **RESTful API:** Fully functional API built with DRF (Django REST Framework).
- **CRUD Operations:** Complete Create, Read, Update, and Delete functionality.
- **Responsive Design:** Built using Bootstrap for a clean UI.

## 🛠️ Tech Stack
- **Backend:** Python / Django 5.x
- **API:** Django REST Framework
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **Database:** SQLite (Dev)

## ⚙️ How to run locally
1. Clone the repo: `git clone [YOUR_URL_HERE]`
2. Create venv: `python -m venv venv`
3. Activate venv: `venv\Scripts\activate` (Windows)
4. Install Django: `pip install django djangorestframework`
5. Run Migrations: `python manage.py migrate`
6. Start Server: `python manage.py runserver`
